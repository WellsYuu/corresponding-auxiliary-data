### 创建订单

```
POST /order/create
```

参数

```
name: "张三"
phone: "18868822111"
address: "慕课网总部"
openid: "ew3euwhd7sjw9diwkq" //用户的微信openid
items: [{
    productId: "1423113435324",
    productQuantity: 2 //购买数量
}]

```

返回

```
{
  "code": 0,
  "msg": "成功",
  "data": {
      "orderId": "147283992738221" 
  }
}

```


### 买家登录

```
GET /login/buyer
```

参数

```
openid: abc
```

返回

`cookie里设置openid=abc`

```
{
    code: 0,
    msg: "成功",
    data: null
}
```

### 卖家登录

```
GET /login/seller
```

参数

```
openid: xyz
```

返回

`cookie里设置token=UUID, redis设置key=UUID, value=xyz`

```
{
    code: 0,
    msg: "成功",
    data: null
```
